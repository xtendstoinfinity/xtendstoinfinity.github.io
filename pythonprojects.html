1. Armstrong number checker:

This program checks if a number is an Armstrong number or not.

ArmstrongNumberChecker.py

print("WELCOME TO ARMSTRONG NUMBER CHECKER")
num = input("Enter a number: ")
length = len(num)
ListOfDigits = list(num)
sum = 0

for digit in ListOfDigits:
    sum += pow(int(digit), length)

if sum == int(num):
    print(num, 'is an armstrong number')
else:
    print(num, 'is not an armstrong number')

Example Output

#Case 1: user enters 12!

> WELCOME TO ARMSTRONG NUMBER CHECKER
> Enter a number: 12
> 12 is not an armstrong number

#Case 2: user enters 153!

> WELCOME TO ARMSTRONG NUMBER CHECKER
> Enter a number: 153
> 12 is an armstrong number

2. Armstrong number generator:

This program generates all possible Armstrong numbers from 0 to 100,000.

print("WELCOME TO ARMSTRONG NUMBER GENERATOR")

    ListOfDigits = list(str(i))

3. Binary-System to Decimal-System converter:

This program converts binary numbers(Eg. 1001) to regular numbers or decimal-system numbers(Eg. 9).

You can also do this by using Python's built-in 'int()' function as: int("Your-Binary-String-Here", 2)

BinaryToDecimalConverter.py

num = input("Enter a binary number: ")
listOfBinaryDigits = list(num)
power_ = len(num)
summedResult = 0

for i in listOfBinaryDigits:
    power_ -= 1
    if int(i) == 1:
        summedResult += pow(2, power_)
    elif int(i) == 0:
        summedResult += 0
    else:
        pass

print(num, "in decimal system:", summedResult)

Example Output

#Example: user enters 10010!

> Enter a binary number: 10010
> 10010 in decimal system: 18

4. Decimal-System to Binary-System converter:

This program converts regular numbers or decimal-system numbers(Eg. 18) to binary numbers(Eg. 10010).

You can also do this by using Python's built-in 'bin()' function as: bin("Your-Decimal-Number-Here")

DecimalToBinaryConverter.py

num = int(input("Enter a number: "))
power = -1
divisorIndex = -1
divisor = 1
binary_value = ""
max = 0

while max <= num:
    power += 1
    max = pow(2,power)
    
n = max/2
temp = num

while (n/divisor) > 1:
    divisorIndex += 1
    divisor = pow(2, divisorIndex)  
    if temp//(n/divisor) == 1:
        binary_value = binary_value + "1"
        temp -=(n/divisor)
    elif temp//(n/divisor) == 0:
        binary_value = binary_value + "0"
    else:
        pass

print(num, "in binary system:", binary_value)

> 27 in decimal system: 11011

5. Fibonacci Series:

This program prints the numbers in the Fibonacci series.

FibonacciSeries.py

series_Length = int(input("How many numbers do you want in the Fibonacci series: "))
temp = 0
previous_Result = 0
current_Result = 1

for i in range(1, series_Length):
    if i == 1:
        print(previous_Result)
    print(current_Result)
    temp = current_Result
    current_Result = previous_Result + current_Result
    previous_Result = temp

This can also be done using Lists and 'append()' function:

FibonacciSeriesUsingLists.py

series_Length = int(input("How many numbers do you want in the Fibonacci series: "))
seriesList = [0,1]

for i in range(0, series_Length):
    seriesList.append(seriesList[-1] + seriesList[-2])
    print(seriesList[i])
